## Overview / 概要

SHOGUN provides **21 MCP (Model Context Protocol) tools** for AI agent integration. These tools allow Claude Desktop and other MCP-compatible agents to read, write, and manage your memory.

SHOGUNは**21のMCP（Model Context Protocol）ツール**を提供し、AIエージェント連携を実現します。Claude Desktopおよび他のMCP対応エージェントからメモリの読取・書込・管理が可能です。

## Quick Reference / クイックリファレンス

### Read Tools

| Tool | Description | Description (JP) |
|---|---|---|
| `get_page` | Get page by slug | slugでページ取得 |
| `list_pages` | List pages with filters | フィルター付きページリスト |
| `search` | Keyword full-text search | キーワード全文検索 |
| `query` | Full hybrid search pipeline | ハイブリッド検索パイプライン |
| `get_timeline` | Get page timeline entries | タイムラインエントリ取得 |
| `get_backlinks` | Get incoming links | 被リンク取得 |
| `traverse_graph` | Traverse link graph | リンクグラフ走査 |
| `get_stats` | Brain statistics | Brain統計 |
| `get_health` | Health report | ヘルスレポート |
| `get_versions` | Version history | バージョン履歴 |

### Write Tools

| Tool | Description | Description (JP) |
|---|---|---|

| `remove_tag` | Remove tag | タグ削除 |
